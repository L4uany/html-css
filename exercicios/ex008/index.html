Citações

A tag usadas para citação é a tag q, que possuí um sentido. Se simplesmente colocarmos aspas, não terá semântica para o site, e no HTML 5, a semântica é fundamental. As aspas são utilizadas apenas em palavras, que não são citações. Exemplo: Sim, ela é "legal" sim.

Exemplo de citação curta: Como disse Humberto Gessinger: Sentir com inteligência, pensar com emoção.

Citação por bloco

O blockquote cria um espaçamento antes da citação, como se fosse um parágrafo, sem aspas. O parâmetro cite depois do blockquote permite colocar o link de onde a citação foi retirada para fins de reconhecimento dos mecanismos de busca, mas não aparece no site em si.
