Principais formatações

Negrito/Destaque, tag B/Strong

Nesta frase temos um termo em negrito utilizando a tag B, do HTML 4 (obsoleta) que representa a forma e não a semântica.

Nesta frase temos um termo em negrito utilizando a tag Strong, com resultado semântico, utilizada no HTML 5.

Itálico/Ênfase, tag I/Em

Nesta frase temos um termo em itálico utilizando a tag i, do HTML 4 (que logo cairá em desuso) que representa a forma e não a semântica.

Nesta frase temos um termo em negrito utilizando a tag em, com resultado semântico, utilizada no HTML 5.

Texto Marcado, tag Mark

Podemos criar também um texto marcado usando a tag Mark, (marca-texto amarelo, mas também podendo mudar a cor através do parâmetro style-CSS, dentro da própria tag. Mark style="background-color:").

Nesse parágrafo, temos outro texto marcado no final.
Se tivermos vários marks e quisermos colocar a mesma configuração para todos, devemos criar uma tag style no head-css, e criar um seletor mark {}, adicionando o parâmetro background-color: cor, em rgba, rba.

Texto grande e pequeno, tag Big/Small

Estamos criando um texto grande e um texto pequeno, utilizando a tag Big e Small. A tag big é forma/não tem sentido e está em desuso, podendo agora ser configurada no CSS, pois é forma e não possuí semântica. Já a tag small continua em uso, pois possuí sentido para estar ali, (observações, em contratos, etc.)

Texto deletado, tag Del

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido (sublinhado), tag Ins/U

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado com a tag U (não semântica)

Texto sobrescrito, tag Sup

Para inserir coisas com o texto em cima, como fórmulas matemáticas. Exemplo: x2+7x-9=0.

Texto subescrito, tag Sub

Para inserir coisas com o texto embaixo, como fórmulas matemáticas. Exemplo: H2O.
