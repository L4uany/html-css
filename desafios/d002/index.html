Favicon + imagem carregada no site

O objetivo desse exercício é escolher um favicon (ícone) e uma foto pro site, considerando o formato do favicon (ico) e o tamanho do pacote do png, para que o site não sobrecarregue.

Resolução:

    
ícone declarado na parte comportamental (head), antes do title, com a tag link:

    <link rel="shortcut icon" href="imagens/favicon.ico" type="image/x-icon">
    
Foto utilizando a tag img:

     <img src="imagens/m.png" alt="Ícone do GIT HUB do Gustavo Guanabara, com boneco gato e corpo de polvo.">
